HTML AND CSS RESOURCE GUIDE

HTML & CSS quick-reference guide that covers all of the HTML tags and CSS properties that I have used for my Recipe Page Website with a description of their purpose and code examples.

HTML (HyperText Markup Language)

is the foundation of any website which provides structure for web pages.

HTML PAGE STRUCTURE (syntax)

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    
</body>
</html>

HTML Tags

01| <!DOCTYPE html>:

Identifies and states the type of document and the version of HTML contained or used in that document to make sure that the web browser will render it properly.

02| <html>:

Defines the root element of the HTML document, which contains all the other HTML elements.

03| <title>:

Defines the title of the HTML document, which is shown in the web browser's title bar or page tab. 

Example: 
 <title>Fluffy Japanese Soufflé Pancake</title>

04| <meta>:

Defines the metadata about the HTML document, which includes the viewport settings and character encoding. 

Example: 
 <meta name="viewport" content="width=device-width, initial-scale=1.0">

05| <link>:

Defines the relationship between the current document and the external resource, which includes the stylesheet. 

Example: 
 <link rel = "stylesheet" type = "text/css" href = "Recipe Page.css">

06| <head>:

Contains the metadata and other head elements that would describe and provide information about the document.

07| <body>:

Contains the text and all other elements that would be displayed in the web page document.

08| <header>:

Defines the introductory content at the beginning of a section or document, which can contain the logos, navigation links, and other contents for the header.

09| <nav>:

Defines a set of navigation links that are used for navigating within the document or to other pages. 

Example: 
 
    <nav class="navbar">
        <a href="#Home">Home</a>
        <a href="#Ingredients">Ingredients</a>
        <a href="#Method">Method</a>
    </nav>

10| <section>:

Defines a section in the document. It is distinct and can be a self-contained block of related content.

11| <div>:

Defines a division in a HTML document. It configures a structural block area with empty space above and below.

12| <h1>:

Used to display the most important heading in a section, which can be the heading of the page or its main title. 

Example: 
 <h1>Fluffy Japanese Soufflé Pancake</h1>

13| <p>:

Displays groups of sentences or sections of text together. 

Example: 
<p>This is one of my go-to pancake recipe, the Fluffy Japanese Soufflé Pancake.</p>

14| <ul>:

Displays an unordered list by using bullet points before each entry in the list. 

Example: 

    <ul> 
        <li>4 Egg whites</li>
        <li>4 Egg Yolks</li>
        <li>4 Tablespoon of Milk</li>
        <li>1 Teaspoon of Vanilla Extract</li>
        <li><sup>1</sup>⁄<sub>2</sub> Cup of All Purpose Flour</li>
        <li><sup>1</sup>⁄<sub>2</sub> Teaspoon of Baking Powder</li>
        <li>4 Tablespoon of Granulated Sugar</li>
    </ul>

15| <ol>:

Displays an ordered list by numbering or lettering to itemize the information contained in the list. 

Example: 

    <ol> 
        <li>Place the 4 egg yolks, 4 tablespoons of milk, and 1 teaspoon of vanilla extract into a large bowl.</li>
        <li>Add the 1/2 cup of all-purpose flour and 1/2 teaspoon of baking powder while whisking and mixing the three ingredients.</li>
        <li>In a separate bowl, place the 4 egg whites and whisk them until firm peaks form, adding four tablespoons of granulated sugar at the same time.</li>
        <li>Mix the two mixtures together.</li>
        <li>Heat up a nonstick pan over low heat and lightly brush with oil.</li>
        <li>Carefully place the mixture in the pan.</li>
        <li>Add drops of water to the pan and cover it with a lid.</li>
        <li>Cook it on low heat for 4-5 minutes on each side.</li>
        <li>Once cooked, remove it and place it on a plate.</li>
        <li>Add any toppings you would like, such as whipped cream, strawberries, honey, maple syrup, butter, or powdered sugar.</li>
        <li>Serve and enjoy immediately!</li>
    </ol>

16| <li>:

Displays a list item within the list. 

Example: 

    <li>4 Egg whites</li>
    <li>4 Egg Yolks</li>
    <li>4 Tablespoon of Milk</li>
    <li>1 Teaspoon of Vanilla Extract</li>
    <li><sup>1</sup>⁄<sub>2</sub> Cup of All Purpose Flour</li>
    <li><sup>1</sup>⁄<sub>2</sub> Teaspoon of Baking Powder</li>
    <li>4 Tablespoon of Granulated Sugar</li>

17| <input>:

Specifies an input field that allows the user to enter data or make selections. 

Example: 

    <li><input type="checkbox"> 4 Egg whites</li>
    <li><input type="checkbox"> 4 Egg Yolks</li>
    <li><input type="checkbox"> 4 Tablespoon of Milk</li>
    <li><input type="checkbox"> 1 Teaspoon of Vanilla Extract</li>
    <li><input type="checkbox"> <sup>1</sup>⁄<sub>2</sub> Cup of All Purpose Flour</li>
    <li><input type="checkbox"> <sup>1</sup>⁄<sub>2</sub> Teaspoon of Baking Powder</li>
    <li><input type="checkbox"> 4 Tablespoon of Granulated Sugar</li>

18| <sup>:

Displays a superscript text. 

Example: 

    <li><input type="checkbox"> <sup>1</sup>⁄<sub>2</sub> Cup of All Purpose Flour</li>

19| <sub>:

Displays a subscript text. 

Example: 

    <li><input type="checkbox"> <sup>1</sup>⁄<sub>2</sub> Teaspoon of Baking Powder</li>

20| <br>:

Creates a line break within the text that also creates a new line without needing to make a new paragraph. 

Example: 
<br>Ingredients</br>

21| <footer>:

Contains the web page document’s footer, which can contain the copyright and authorship information. 

 Example:
 <footer> 
        <div class="footer">
            &copy; | Made by: Isaac Gerard S. Marquez
        </div>
    </footer>


CSS (Cascading Style Sheets)

is a style sheet language used to style and format Web pages.

CSS STRUCTURE (syntax)

Selector {
    Property 1 : value;
    Property 2 : value;
    Property 3 : value;
}

CSS Properties

01| {width}

Sets the width of an element. 

    Example:
.element{
   width: 100%;
} /* I set the width to 100% to make it span the entire width. */

02| {height}

Sets the height of an element.

Example:
    .element{
   height: 600px;
} /* I set the height to 600 pixels. */

03| {background}

Sets the background properties of an element.

Example:
    .element{
   background: url('Recipe Images/Pancake.jpg') no-repeat center / cover;
} /*I set the background image to "Pancake.jpg". It is center positioned and covers the entire area.*/

04| {background-color}

Sets the background color of an element.

Example:
    .element{
   background-color: #2f3d3c;
} /* Sets the background color of the element to a color with the hex value #2f3d3c. */

05| {min-height}

Sets the minimum height of an element.

Example:
    .element{
   min-height: 400px;
} /* I set the minimum height of the element to 400 pixels. */

06| {border-radius}

Rounds the corners of an element's border.

Example:
    .element{
   border-radius: 8px;
} /* I set the border radius to 8 pixels to apply rounded corners to the element. */

07| {display}

Sets or specifies the display behavior of an element.

Example:
    .element{   
display: flex;
} /* To create a flexbox layout. */

08| {flex-direction}

Sets the direction of the flexible items in a flex container.

Example:
    .element{
   flex-direction: column;
} /* To display contents in the column. */

09| {justify-content}

Aligns the flex items along the main axis of the current line of the flex container.

Example:
    .element{
   justify-content: center;
} /* Aligns the element in the center. */

10| {padding}

Sets the padding area for all the four sides of an element.

Example:
    .element{
   padding: 50px;
} /* Adding 50 pixels of padding to all the four sides of the element. */

11| {padding-left}

Sets the left padding of an element.

Example:
    .element{
   padding-left: 0;
} /* I used the padding-left: 0 to remove any default left padding. */

12| {margin-top}

Sets the top margin of an element.

Example:
    .element{
   margin-top: 50px;
} /*For spacing, I added a top margin of 50 pixels.*/

13| {margin-left}

Sets the left margin of an element.

Example:
    .element{
   margin-left: -150px;
} /*To move the element to the left, I set the margin-left to -150 pixels.*/

14| {margin-bottom}

Sets the bottom margin of an element.

Example:
    .element{
   margin-bottom: 20px;
} /*For spacing, I added a bottom margin of 20 pixels.*/

15| {color}

Sets the color of the text.

Example:
    .element{
   color: white;
} /* To set the text color to white.*/

16| {max-width}

Sets width of an element to maximum.

Example:
    .element{
   max-width: 1000px;
} /* To set the maximum width of the element to 1000 pixels.*/

17| {font-size}

Sets the size of the font.

Example:
    .element{
   font-size: 20px;
} /* To set the size of the font or text to 20 pixels.*/

18| {text-align}

Sets the horizontal alignment of text content.

Example:
    .element{
   text-align: center;
} /* To center the text element.*/

19| {list-style-type}

Sets the appearance of a list item marker.

Example:
    .element{
   list-style-type: disc;
} /*To show the bullet points for each item from the list, I set the list-style-type property to disc.*/

20| {align-items}

Sets the default alignment for items inside a flex container.

Example:
    .element{
   align-items: center;
} /* To align the elements at the center.*/
